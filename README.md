# Escampe
![Escampe](https://github.com/Poulpy/escampe/blob/master/escampe.png?raw=true)


## Clone the project

```bash
git clone https://github.com/joshhauser/Escampe.git
```

## Dependencies

- SDL:
  - libsdl1.2-dev
  - libsdl-ttf2.0-dev
  - libsdl-ttf2.0-0
- Freetype:
  - libfreetype6
  - libfreetype6-dev

## Compile
```bash
make
```

## Execute

```bash
./bin/main
```
